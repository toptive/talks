What is GraphQL?

GraphQL es un nuevo API standard que provee una alternativa más eficiente, poderosa y flexible a REST

Some things about REST

  • Los recursos son definidos por Endpoints
  • El Cliente puede llamar a muchos endpoints
  • Usa HTTP como protocolo de comunicación
  • Conceptualmente simple
  • Completamente probado

    • GraphQL is the better REST

    En la última decada, REST se convirtio en el standard para el diseño de web APIs.

    Sin embargo, las API REST han demostrado ser demasiado inflexibles para mantenerse al día con los rápidos cambios en los requerimientos de los clientes que acceden a ellas.

    GraphQL resuelve muchas de las deficiencias e ineficiencias que los desarrolladores experimentan al interactuar con API REST.
